Task type project IDs

OCR regions

Keyword Highlights

Matched strings are highlighted in ChatGPT messages. Adjacent terms are included before or after the match within the selected range. Normal strings ignore punctuation and hyphens; r- regex lines run against the raw text so symbols, commas, and decimals still work. Prefix a matched-string line with -- to exclude it from that rule.
